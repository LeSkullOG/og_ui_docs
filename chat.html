Chat Skin ​

OG_UI includes a themed chat skin that can render normal FiveM chat messages, command suggestions, and a focused input bar.

Use /ogchat locally to preview the feed, suggestions, and input.

Client Exports ​

lua
exports.og_ui:AddChatMessage({
    type = 'dispatch',
    author = 'Dispatch',
    template = 'dispatch',
    title = 'Dispatch Call',
    subtitle = 'Legion Square',
    meta = 'Code 2',
    text = 'Suspicious activity reported near Legion Square.'
})

exports.og_ui:AddChatSuggestion('/me', 'Send a roleplay action.', {
    { name = 'action', help = 'What your character does' }
})

exports.og_ui:OpenChatInput()

Message Fields ​

FieldTypeNotes
typestringVisual channel. Common values: player, system, staff, dispatch, ooc, me, do, tweet, ad.
authorstringOptional sender or source label.
textstringMessage body.
labelstringOptional display label if you do not want the default channel label.
argstableCompatible with standard chat:addMessage style payloads.
templatestringOptional richer layout. Current values: dispatch, staff, announcement.
titlestringOptional template headline.
subtitlestringOptional template subheading.
metastringOptional small right-aligned template metadata.

Standard Events ​

The chat skin listens for these client events:

lua
TriggerEvent('chat:addMessage', {
    type = 'staff',
    author = 'Admin',
    text = 'Restart warning in five minutes.'
})

TriggerEvent('chat:addSuggestion', '/report', 'Send a staff report.', {
    { name = 'message', help = 'Report details' }
})

TriggerEvent('chat:addSuggestions', {
    {
        name = '/me',
        help = 'Send a roleplay action.',
        params = {
            { name = 'action', help = 'What your character does' }
        }
    }
})

TriggerEvent('chat:clear')

Config ​

lua
Config.Chat = {
    enabled = true,
    bindKey = false,
    disableNative = true,
    key = 'T',
    placeholder = 'Type a message or /command',
    maxLength = 240,
    defaultChannel = 'local',
    includeRegisteredCommands = true,
    requireAceForRegisteredCommands = false,
    suggestionRefreshMs = 30000,
    commandSuggestionBlacklist = {
        toggleChat = true,
        og_chat_open = true
    },

    channels = {
        ['local'] = {
            label = 'Local',
            type = 'player',
            color = '#d6a84f',
            command = nil,
            help = 'Send a local chat message.'
        },
        me = {
            label = 'Me',
            type = 'me',
            color = '#4ade80',
            command = 'me',
            prefix = '*',
            help = 'Send a roleplay action.'
        },
        ooc = {
            label = 'OOC',
            type = 'ooc',
            color = '#c084fc',
            command = 'ooc',
            help = 'Send an out-of-character message.'
        },
        staff = {
            label = 'Staff',
            type = 'staff',
            color = '#f87171',
            command = 'staff',
            ace = 'og_ui.chat.staff',
            target = 'ace',
            template = 'staff',
            help = 'Send a staff-only message.'
        }
    }
}

disableNative hides the bottom-right native fallback chat that can appear when the default chat resource is stopped. bindKey is disabled by default to avoid double-opening if another chat resource is still mapped to T. Once OG_UI owns chat, stop/disable the other chat resource and set bindKey = true.

Channel Config ​

Server owners can add, remove, or rename channels in Config.Chat.channels.

FieldTypeNotes
labelstringBadge text shown in the chat message.
typestringCSS class suffix used for fallback styling.
colorstringOptional accent color for the message stripe and label.
commandstring/nilSlash command routed into this channel, without /. Use nil for the default plain text channel.
prefixstringOptional prefix added to the displayed message, useful for /me.
helpstringSuggestion text shown while typing commands.
paramHelpstringOptional suggestion helper for the message parameter.
acestringOptional ACE permission required to use/see this channel suggestion.
targetstringOptional delivery mode: all, source, ace, or nearby.
rangenumberDistance used when target = 'nearby'.
templatestringOptional chat message template for this channel.

When a player submits /me waves, OG_UI finds the channel whose command = 'me', sends it through the server route, and broadcasts it using that channel label/color. If the slash command is not configured as a chat channel, OG_UI falls back to ExecuteCommand.

For private channels, set ace and target = 'ace'. For example /staff hello will only send to players who are allowed for og_ui.chat.staff, and unauthorized players cannot submit to that channel.

Normal chat messages fade out when the chat input is closed. Opening chat shows the recent history again, controlled by the player settings for visible messages and fade delay.

Command Suggestions ​

OG_UI pulls registered FiveM commands using GetRegisteredCommands() on both the client and server, similar to the default chat resource. Commands from other resources, such as /restart, can appear while typing even when they are not listed in Config.Chat.channels.

Set requireAceForRegisteredCommands = true if you only want to show registered command suggestions when IsPlayerAceAllowed allows command.<name>. This only controls suggestions; FiveM still handles command permissions when the player submits the command.

suggestionRefreshMs controls how often OG_UI refreshes registered command suggestions after the first sync. Keeping this at a sensible value avoids re-sending every command each time chat opens.

Use commandSuggestionBlacklist to hide internal or unwanted commands from autocomplete. Pressing Tab completes the first visible suggestion instead of moving browser focus. Pressing Up recalls the last sent message, and Down moves back through the draft/history.

Suggestions with params are rendered as command signatures, so this:

lua
TriggerEvent('chat:addSuggestion', '/report', 'Send a staff report.', {
    { name = 'message', help = 'Report details' }
})

appears as /report [message] in the suggestion list and active command preview.
