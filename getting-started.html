Install & Build ​

Installation ​

Add og_ui to your server resources and ensure it starts before any resource that calls its exports.

text
ensure og_ui
ensure your_resource

The FiveM runtime loads the built UI from dist/index.html. Frontend source lives in web.

Frontend Build ​

Only needed after editing files in web/src.

powershell
cd web
npm.cmd install
npm.cmd run build

The build writes directly to dist, which is what FiveM serves through fxmanifest.lua.

Runtime Options ​

Most UI open calls accept these focus/input options.

OptionTypeDefaultDescription
lockMovementbooleanfalseDisables walking, sprinting, jumping, aiming, firing, and melee while the UI is open.
captureMousebooleantruePasses mouse focus to NUI. Set false for keyboard/controller-only UIs.
keepInputbooleanfalseAllows game input to continue while NUI is focused. Useful for lightweight context menus.

Common window options:

OptionTypeNotes
titlestringWindow title.
subtitlestringOptional secondary text.
positionstringCommon values: center, right.
sizestringCommon values: sm, md, lg, xl.
variantstringCurrent common value: glass.

Server Defaults ​

Server owners can control default player settings in shared/config.lua.

lua
Config.Settings = {
    enabled = true,

    defaults = {
        theme = 'gold',
        soundsEnabled = true,
        soundVolume = 80,
        reducedMotion = false,
        density = 'standard',
        notificationPosition = 'top-right',
        progressPosition = 'bottom-center'
    },

    allowPlayerOverride = {
        theme = true,
        sounds = true,
        soundVolume = true,
        reducedMotion = true,
        density = true,
        notificationPosition = true,
        progressPosition = true
    }
}

Config.Chat = {
    enabled = true,
    bindKey = false,
    disableNative = true,
    key = 'T',
    placeholder = 'Type a message or /command',
    defaultChannel = 'local',
    channels = {
        ['local'] = {
            label = 'Local',
            type = 'player',
            color = '#d6a84f'
        },
        me = {
            label = 'Me',
            type = 'me',
            color = '#4ade80',
            command = 'me',
            prefix = '*'
        }
    }
}

If an override is set to false, the settings menu shows that option as locked and keeps the server default active. If enabled is false, OpenSettings shows a warning notification instead of opening the settings window.
