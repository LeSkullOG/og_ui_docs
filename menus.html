Menus & Dialogs ​

Notifications ​

lua
exports.og_ui:Notify({
    title = 'Vehicle Locked',
    description = 'Your vehicle has been locked.',
    type = 'success',
    duration = 3500
})

Action buttons can be added for lightweight prompts such as dispatch calls, invites, route requests, and trade notices.

lua
exports.og_ui:Notify({
    id = 'dispatch_call_42',
    title = 'Dispatch Call',
    description = 'Suspicious activity reported near Legion Square.',
    type = 'info',
    duration = 9000,
    actions = {
        { label = 'Accept', value = 'accept', variant = 'primary' },
        { label = 'Route', value = 'route' },
        { label = 'Decline', value = 'decline', variant = 'danger' }
    }
})

AddEventHandler('og_ui:notificationAction', function(data)
    if data.notificationId == 'dispatch_call_42' and data.value == 'accept' then
        -- Accept call here
    end
end)

Notification fields:

FieldTypeNotes
titlestringMain notification text.
descriptionstringOptional body text.
typestringinfo, success, warning, or error.
durationnumberDuration in milliseconds.
actionstableOptional action buttons. Up to three are shown.
buttonstableAlias for actions.

Notification action fields:

FieldTypeNotes
labelstringVisible button text.
valueanyStable action identifier returned in the event payload.
variantstringprimary, muted, success, or danger.
disabledbooleanPrevents the action from being clicked.
closebooleanSet false to keep the toast open after click. Defaults to closing.

Standard Menu ​

Use OpenMenu for larger interactive systems such as jobs, shops, crafting, emotes, and admin tools.

lua
exports.og_ui:OpenMenu({
    title = 'Mechanic Menu',
    subtitle = 'Vehicle actions',
    position = 'right',
    lockMovement = true,
    items = {
        {
            label = 'Repair Vehicle',
            description = 'Fix the closest vehicle.',
            value = 'repair',
            icon = 'Wrench'
        },
        {
            label = 'Clean Vehicle',
            value = 'clean',
            icon = 'Sparkles'
        }
    }
})

Use OpenMenuAwait when the calling script should wait for a selected menu item.

lua
CreateThread(function()
    local result = exports.og_ui:OpenMenuAwait({
        title = 'Garage',
        items = {
            { label = 'Spawn Sultan', value = 'sultan' },
            { label = 'Spawn Banshee', value = 'banshee' }
        }
    })

    if result and result.ok then
        print(result.value)
    end
end)

Context Menu ​

Use OpenContext for lightweight movement-friendly prompts.

lua
CreateThread(function()
    local result = exports.og_ui:OpenContext({
        title = 'Vehicle',
        subtitle = 'Nearby actions',
        captureMouse = false,
        keepInput = true,
        options = {
            { label = 'Lockpick', value = 'lockpick' },
            { label = 'Inspect VIN', value = 'vin' }
        }
    })

    if result and result.ok then
        print(result.option.value)
    end
end)

Context menus are intended to stay arrow-key/controller driven. WASD is left to the game when keepInput = true.

Confirm Dialog ​

lua
CreateThread(function()
    local result = exports.og_ui:OpenConfirm({
        title = 'Delete Vehicle',
        description = 'Are you sure?',
        confirmText = 'Delete',
        cancelText = 'Cancel',
        lockMovement = true
    })

    if result and result.confirmed then
        -- Continue
    end
end)

Input Dialog ​

lua
CreateThread(function()
    local result = exports.og_ui:OpenInput({
        title = 'Register Business',
        fields = {
            {
                name = 'business_name',
                label = 'Business Name',
                type = 'text',
                placeholder = 'Enter a name'
            }
        }
    })

    if result and result.submitted then
        print(result.values.business_name)
    end
end)

Input fields:

FieldTypeNotes
namestringRequired key used in result.values.
labelstringVisible label.
typestringInput type, for example text or number.
placeholderstringPlaceholder text.
defaultValuestringOptional initial value.

Progress ​

lua
local started = exports.og_ui:Progress({
    title = 'Searching Vehicle',
    description = 'Please wait...',
    duration = 5000,
    variant = 'linear'
})

if started then
    AddEventHandler('og_ui:progressComplete', function()
        -- Completed
    end)
end

Progress returns false if another progress UI is already active.
