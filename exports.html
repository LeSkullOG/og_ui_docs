Result Shape ​

Result-based exports return tables with a shared shape.

FieldTypeNotes
okbooleantrue when the UI resolved with a positive or selected result.
cancelledbooleantrue when the UI was cancelled or closed without a positive result.
typestringUI type, for example menu, context, confirm, input, grid, or list.
idstringRequest id generated by OG_UI or supplied by the caller.

Legacy fields are still preserved, such as confirmed, submitted, option, and values.

Utility Exports ​

lua
local isOpen = exports.og_ui:IsOpen()
local depth = exports.og_ui:GetStackDepth()
local topComponent = exports.og_ui:GetTopComponent()

exports.og_ui:CloseTop()
exports.og_ui:CloseAll()
ExportDescription
IsOpenReturns true when any OG_UI window is open.
GetStackDepthReturns the current Lua-side UI stack depth.
GetTopComponentReturns the active component name, such as menu, context, confirm, input, or settings.
CloseTopCloses only the top UI window and cancels it if it is waiting for a result.
CloseAllCloses every OG_UI window and cancels all pending results.
CloseMenuLegacy alias for closing all OG_UI windows.

Chat Exports ​

ExportDescription
AddChatMessageAdds a styled chat message to the OG_UI chat feed.
ClearChatClears the local chat feed.
AddChatSuggestionAdds a command suggestion shown while typing /commands.
RemoveChatSuggestionRemoves a command suggestion by name.
OpenChatInputOpens the focused chat composer.

Events ​

OG_UI emits these client events.

EventPayloadDescription
og_ui:itemSelecteditemFired when a standard menu item is selected.
og_ui:hoverItem{ item, category }Fired when the highlighted menu item changes.
og_ui:notificationAction{ id, notificationId, action, value }Fired when a notification action button is clicked.
og_ui:contextSelect{ id, option }Fired when a context option is selected.
og_ui:gridSelect{ ok, item, action, value, actionValue }Fired when a grid item/action is selected.
og_ui:listSelect{ ok, item, action, value, actionValue }Fired when a list record/action is selected.
og_ui:chatSubmitmessageFired locally when the OG_UI chat input submits a message.
og_ui:confirmResult{ confirmed, id, value }Fired when a confirm dialog resolves.
og_ui:inputResult{ submitted, id, values }Fired when an input dialog resolves.
og_ui:progressCompletenoneFired when progress completes.
